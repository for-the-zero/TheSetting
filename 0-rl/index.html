世界

原本世界是一个七维正方体，内部由世界素组成

来自异世界泄露的“时间”打破平静，将世界分割为不同的小块

但这玩意很巧，分割后成为世界块，随着时间运动却不会相撞

知道两个“时间”相撞，推动一块世界块与另一块相撞

挤压时发生反应，在内部发生爆炸，这一炸，炸出了整个宇宙

随后宇宙往中心不断塌陷，同时宇宙边缘吸收世界素和“时间”

将世界素拆分后转为宇宙粒子向内飞入，同时会被“时间”分割

塌陷的同时所有东西都会缩小，但总空间不变

对于里面来说，它在扩大，实际上是自己在缩小

同时也在补充外来的世界素

宇宙中心是宇宙的“时间”核心，极其小

但一旦到达那里，就会化为“时间”，不复存在

注：

世界素：形状为多根细线（相对我们很粗）螺旋旋转，并类似于纤维，世界的一种组成部分

世界块：世界这个方块一样的形状和性质，但是不能叫它们为单独一个世界，因为它们互相联系

宇宙粒子：构成宇宙的基本粒子

“空间”

在世界之外，所有世界处于“空间”之中

这是观测到最大的层次，其次是世界

空无一物，比太空中的真空还空

“时间”

“时间”是一把无形的刀，可以切割任何物质

在自我切割时会分裂，同时产生一定的力，与自身的大小有关

当它对世界素切割够细时，将变为宇宙粒子，无法再切割

切割无法切割的物质，可以将它的体积变小，同时对运行造成影响

被影响的运行，会被拉长，成为我们熟知的时间

时间循环

对于我们熟知的时间，它是一直循环的，但它分为三种状态：微循环、框架循环、总循环

这个循环不同于普通循环，有不确定性(产生变异)和嵌套性

每天几点起床睡觉吃饭学习工作都是差不多时间的，每周的生活都是差不多结构的，每年都会在某个时候有什么节日，但以上的内容每一次循环都有不同

由于“时间”来自异世界，出现原因待研究

微循环

即一个很小的循环

这种循环的感受很小，也就是循环度小，以至于可以忽视，它每次循环改变的东西很多

框架循环

即一个区间的循环

例如今天过完了就是第二天，第二天一样过，这就是以“天”为框架的循环

每一天都有不同，而且每次循环都会影响下一次循环，循环度较小

当然也有更细分的，但是大多比较接近微循环，就不讲了

而“周”为框架的循环也包括“天”为框架的循环，有时候，循环度更高，但是有几天有变异的存在，会与其它几天不同

如此下来，什么都有

每周都有重复的，有的时候又不一样

每年12个月都差不多重复的，也有些不一样

总循环

最大的循环，当世界进行到底时触发，一切归零，重新开始，没有更大的循环

变异

有时候循环并不是一切不变的，会发生变异，使每次循环都改变，都不一样

产生原因和变异方向等有待研究

微变异

一个框架内发生的小型变异，无关紧要，变异内容＜10%

内部变异

在一个循环框架内的框架循环发生个体变异时即为内部变异

即框架内部的变异

内部发生变异时可能不止1个

有时不会变异，有时可能全部变异

个体变异

即整个框架的变异

整个框架变异程度极大，几乎完全不同

循环变异

一次变异并非偶然，在循环框架出现个体变异后，母循环框架被再次循环，这次出现相同变异的可能性极大

物体和环境都是由宇宙粒子组成

不同强度的力按不同的方式结合，会有不同的特性

最常见的成为夸克，但还有很多尚未观测到

拥有很多类型，有的可以独立完成一些工作，有的需要连接起来组成具有一定功能的物质，几乎无处不在，但含量极少，不如夸克多

粒子射线

射线的功能要看是什么粒子，有的能对周围造成影响，有的能获取周围信息

宇宙法则

实体与虚体

宇宙之内存在两种状态

实体

由宇宙粒子组成的物质，有占用空间

且由本身性质直接工作运行，无需其它实体帮助

虚体

由宇宙粒子等物质存储，以其它实体进行转化才能工作运行

与程序、模拟器等的概念类似

毁灭核心

万物皆有一个点，可以支撑整个实体运转

它就是毁灭核心，可以是脆弱的，可以是坚固的

但一旦收到一点点损伤，都会使对应实体毁灭

由一种宇宙粒子组构成，架构类似，构成它的粒子越多越坚固

当它接收到的能量大于它能接受的量，将造成损伤，即毁灭

一个空间也可以有，空间内的物体也可以有，但空间的毁灭会波及内部

宿命

在生物的控制中枢，会有一个特殊的标记排布在不同的细胞中

不是所有的细胞都有，拥有的量也不一样

每个生物中的排列方式都不一样

通过长期研究初步解码，疑似预示着未来会发生的事

因此，有一个推论：

这是每个生物的宿命

记载着它们未来会拥有什么、会失去什么

记录着重要的节点，虽然有变量，但一定会发生的是一定会发生

这就是宿命，没人能逃脱

在大量的实验中也证明了这点

（目前不知道这种现象是世界级别的还是宇宙级别的，先放宇宙法则中）

因果律动

一个可以控制宿命的东西，但本身难以控制

有两个部分：律动谐音、律动火花

  • 律动谐音：主要对应和谐的东西

  • 律动火花：主要对应冲突等东西

它们在宇宙的分布是均匀的，以微量宇宙粒子的形式存在与宇宙之中，拥有很强的粒子射线能力，体积都占宇宙的85%，重复的70%就是两种力量冲突而融合的结果

它们可以在宇宙中流动，分布并不均匀，分布的密度代表力量的强弱，区域受到影响，事件可以被改变，甚至可以改变宿命的执行

对冲律动

两个部分在重复的地方冲突融合，形成对冲流

可以使事件发展不定向

或是使事件脱离这个衡量标准——和谐与冲突，而是往其它方向发展

但这并不是紊乱

另外，对冲时力量并不均衡，而是有高有低，这会使事件的发展更偏向于更高的一方

生命/思绪

细胞内部拥有较多但是很小的粒子组，带给细胞“基础生命”，主要在细胞核

而在大脑的神经元中它更多，有微量的临时处理单元，通过组合方式的不同有各自擅长的功能

靠近的细胞，粒子组会连接并适应，适应会改变形态，“基础生命”的智慧也会1+1大于2

死亡

生命体年老的时候改变停不下，适得其反导致中心失去控制能力，从而导致细胞死亡

或者在物理层面导致细胞死亡后，适应形成的重要连接点断开，导致范围内细胞失常，蝴蝶效应影响到周围，使大脑无法工作，(至少)无法操控生命体，人就死了

进化

而生命在进化，包括是因为适应过程中对胎儿有影响，如今内部的粒子已经发生了极大的变异，成为如今生命构造的绝版粒子，当初的粒子只能构造未进化的生命

另外其他物体(甚至包括温度、光)也是由宇宙粒子组成的，但是并未形成实体，也存在着“适应”，当遇到其他粒子组时反应更强烈，一部分形成化学反应

当粒子组“适应”时错误变异，动变静可能会形成肿瘤甚至非细胞结构物质，只是概率少之又少，几乎没遇到过；静变动在合适的条件下可以形成原始生命，幸运的话有概率长出植物甚至生物

记忆

记忆部分的神经元拥有额外且中性(无其他性质)的结构，它易于因为适应而改变，形成记忆核心，存储着记忆片段，读取和写入时旁边的粒子组受到刺激放出粒子光照射修改形状，而且不会改变其中性的状态

在母体里出生时，会有部分记忆通过浓缩后传输到胎儿，但胎儿并未察觉，需要触发

负责传输的是一些隐藏物质，极小，大概是由某种宇宙粒子组成的

浓缩后传输到胎儿，在胎儿体内运动到对应的地方再释放出来

触发记忆需要一个前提，叫键(key)，被触发的叫值(value)，类似与计算机的字典类型

通过一定的情景作为键触发值获取记忆并传输，但长段记忆是分段的，以值的末尾作为键查找下一段记忆

但是不同的是一个键不一定对应一个值，需要一段思考去细化筛选条件寻找最符合的记忆片段

触发未触发的记忆后，记忆会向告诉读写区移动，那里读写功能更发达，能记起但容易忘

慢慢记起来了，再往那个方向移动进入读取发达区，不容易被覆盖，也就是记牢了

但是当我们回忆的时候，会觉得时间很快，因为在记忆时我们不会吧所有东西都记下来，会忽略部分内容，就说记下来了也可能忘记，所以留下来的记忆就不多，再加上回忆的时候不能全部出发所有对应的键，内容就很少，时间就很短，就会觉得时间过得很快
